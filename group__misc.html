enum exword_error

Error codes.

Most libexword functions return 0 on success or one of these error codes.

Enumerator:
EXWORD_SUCCESS 

Success (no error)

EXWORD_ERROR_FORBIDDEN 

Access denied.

EXWORD_ERROR_NOT_FOUND 

File not found.

EXWORD_ERROR_INTERNAL 

Internal Error.

EXWORD_ERROR_NO_MEM 

Insufficient memory.

EXWORD_ERROR_OTHER
